# BTG Funds App

Aplicación de gestión de fondos de inversión (Prueba Técnica).

## Stack Tecnológico
- Angular 18
- TailwindCSS
- JSON Server (Mock API)

## Estructura del Proyecto
- `src/app/core`: Servicios y Modelos (Singleton).
- `src/app/features`: Módulos funcionales (Fondos, Historial).
- `src/app/shared`: Componentes reutilizables.
